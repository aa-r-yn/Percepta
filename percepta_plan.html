Risks to manage
High
False positives — umbrellas, pens flagged as knives. Audit this hard in Week 3 before building anything on top of the model.
High
GPU access stall. Without it Weeks 2–3 collapse. Confirm and run a test training job in Week 1 before anything else.
Medium
Dataset class imbalance — knife images likely outnumber handgun. Check distribution on merge and use augmentation to balance.
Medium
Scope creep. Abandoned object detection and restricted zone intrusion are post-MVP only if Weeks 1–6 go cleanly.
